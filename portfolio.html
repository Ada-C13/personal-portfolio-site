Slack CLI Application

Built a command line application for Slack in Ruby that allows the user to view information about a Slack workspace as well as interact with it. Hotel

Built a hotel booking system in Ruby that allows the user to manage reservations. The application allows the user to create reservations, either in single or in a block. The user is also able to search through listings by date and by room.
